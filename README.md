A live, composable user interface, where users and developers can customize model setting views while the app is running. This capability enables many UI changes to be made without any code changes.

## Details

A "customize settings" UI is provided to enable reordering model setting views, toggling visibility, and renaming setting labels, where names are retrieved from a live localized string bundle that is managed in the "Application Support" directory.

Users can also choose from a variety of viewing styles, including adaptive layout, compact or expanded layouts, or create their own custom layouts with specific combinations of text and icon labels and various forms of control surfaces.
